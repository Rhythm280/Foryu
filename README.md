# Foryu

A romantic, AI-powered gift page built with Next.js 14, TypeScript, Tailwind CSS, Framer Motion, and Anthropic Claude.

## What this project is

This app lets someone create a personalized gift page for their partner with a custom poem, heartfelt message, soft motion, and shareable artwork.

## Run locally

1. Install dependencies
   ```bash
   npm install
   ```
2. Create `.env.local` from `.env.example`
3. Add your Anthropic API key to `.env.local`
4. Start the app
   ```bash
   npm run dev
   ```

## Anthropic API key

Get your Anthropic API key from the Anthropic dashboard and set it as `ANTHROPIC_API_KEY` in `.env.local`.

## Deploy to Vercel

1. Push the repo to GitHub
2. Import the project in Vercel
3. Add `ANTHROPIC_API_KEY` to Vercel environment variables

## Customize colors and content

- Accent pastel options are defined in `app/page.tsx` and styled in components.
- The gift experience is generated by the Claude prompt in `app/api/generate/route.ts`.
- The art canvas behavior is driven by `components/ArtCanvas.tsx` and its `artSeed` values.
